== Description ==

My Twitter is a WordPress plugin made by Xhanch Studio to show your latest tweets, replies, user's profile picture(or you may just show the name instead of picture), post date(changeable format), click-able links within the tweets, convert smilies syntax to smilies and some custom texts(can be HTML code). My Twitter generates very clean, table-less and valid XHTML codes.
This plugin use NO JavaScript.
